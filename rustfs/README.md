# RustFS Compose

Docker Compose configuration for [RustFS](https://github.com/rustfs/rustfs) - an S3-compatible object storage system.

## Ports

| port   | info    |
| ------ | ------- |
| `9000` | S3 API  |
| `9001` | Console |

Console credentials:

```txt
username: rustfsadmin
password: rustfsadmin
```
